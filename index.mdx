      Get up and running with Restack CLI in minutes
    </p>

  <Steps>
    <Step title="Install Restack CLI">
      Clone the repository and link the CLI tool globally:
      
      ```bash
      git clone https://github.com/uglyus-daniel/restack-cli.git
      cd restack-cli
      npm link
      ```
    </Step>

    <Step title="Identify your commit range">
      Find the start and end commits you want to reorganize:
      
      ```bash
      git log --oneline
      ```
      
      Note the commit hashes for the first and last commits in your range.
    </Step>

    <Step title="Run the restack command">
      Execute the command with your commit range and base branch:
      
      ```bash
      restack <start-commit> <end-commit> [base-branch]
      ```
      
      The tool will analyze your commits, detect PR boundaries, and reorganize them into clean branch chains.
    </Step>

    <Step title="Review and push">
      After the restack completes, review the new branches and optionally push them to remote:
      
      ```bash
      git log --oneline --graph --all
      ```
      
      The tool will prompt you to push all branches with `--force-with-lease` for safety.
    </Step>
  </Steps>
</div>
